# New Journey Flow: Q1-Q5 Structure

**Date:** 2026-02-19
**Source:** Storyboard (new-flow1 through new-flow17), journey-specification.md, feedback-v6.md
**Status:** Draft - pending clarification

---

## Overview

The presentation is a **two-part experience**: slides + interactive map. The map activates at specific cue points during the slide narrative. This document captures the **map-only portions** since slides are handled separately.

Two entry paths:
1. **Primary:** Guided Q1-Q5 journey (presenter-led, sequential)
2. **Secondary:** "I've seen this before" - fully loaded interactive map

---

## Slide-to-Map Transition Cues

| Cue | Trigger | Direction |
|-----|---------|-----------|
| Resource list (water/power/strategic) | Slide showing 3 resources | Slides -> Map |
| Airline map route | Strategic location complete | Map -> Slides |
| Pipeline slide | Talent pipeline slide shown | Slides -> Map |
| Future switch click | Future outlook toggle | Stays on Map |
| QA section | Q&A begins | Slides -> Map (final) |

- Right panel: screenshot of JASM ESG report as evidence

### Power (3 sub-steps, similar to talent pipeline pattern)
- **Solar:** markers across Kyushu + evidence on right panel + connection lines to JASM
- **Wind:** markers across Kyushu + evidence on right panel + connection lines to JASM
- **Nuclear:** markers (Genkai, Sendai) + evidence on right panel + connection lines to JASM

- Airline-style route map showing Kyushu position relative to Taiwan, Korea, Shanghai
- **This is the cue to switch back to slides**
